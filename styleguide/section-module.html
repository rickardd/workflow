14 #module Module

Module is a container for basic content.

.module--margin adds a margin beneath the module

Modules are often combined with module-content blocks. Please navigate to module-content section for more info.

14.11 #module.tabs Module-tabs

14.11.1 #module.tabs.1 Module tabs

Basic Tab Structure Tabs sit atop a .module.module--white element

Widths are automatic Upto 5 tabs Use text-hide classes such as .hide-mob on tab content elements to make responsive States:

The class .is-indent will indent the tabs 24px on either side.

Tabs

Tabs have two states:

.is-active is the class to be added on the active tab to change the color of a tab, add the state class .is-red to the .module-tab (colors include is-red,is-yellow,is-blue,is-green)
