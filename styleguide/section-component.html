9 #component Component

A component is a mix of elemnts. E.g a drop down which is using a button and list style.

9.1 #component.breadcrumb Breadcrumbs

Breadcrumbs are primarly used for navigation purposes when a back action is needed.

9.1.1 #component.breadcrumb.1 General

Breadcrumbs have a max width of 33%, add a .text-ellipsis class to text inside so that is formats nicely Note that the last item is a p tag

Example
  • One
  • Open
  • Face

  • Sandwich

<ul class="breadcrumb clearfix">
  <li>
    <a href="#" class="text-ellipsis">
      One
      <span class="flat-icon icon-16 arrowright-icon"></span>
    </a>
  </li>
  <li>
    <a href="#" class="text-ellipsis">
      Open
      <span class="flat-icon icon-16 arrowright-icon"></span>
    </a>
  </li>
  <li>
    <a href="#" class="text-ellipsis">
      Face
      <span class="flat-icon icon-16 arrowright-icon"></span>
    </a>
  </li>
  <li>
    <p class="text-ellipsis">Sandwich</p>
  </li>
</ul>

9.1.2 #component.breadcrumb.2 Heading breadcrumb

Add .breadcrumb-heading class to the ul Also change the icon-size to .icon-32 from .icon-16

Example
  • One
  • Open
  • Face

  • Sandwich

<ul class="breadcrumb breadcrumb-heading clearfix">
  <li>
    <a href="#" class="text-ellipsis">One<span class="flat-icon icon-32 arrowright-icon"></span></a>
  </li>
  <li>
    <a href="#" class="text-ellipsis">Open<span class="flat-icon icon-32 arrowright-icon"></span></a>
  </li>
  <li>
    <a href="#" class="text-ellipsis">Face<span class="flat-icon icon-32 arrowright-icon"></span></a>
  </li>
  <li>
    <p class="text-ellipsis">Sandwich</p>
  </li>
</ul>

9.3 #component.loading-region Loading region

The loading reggion will expand the loading area to it's chilldrens height.

Example
If this content expands or shrink the spinner will change with it.
<div class="module__body">
 <div class="loading-region">
  <div class="module-content module-content--padding-all">
   <div style="height: 500px"> If this content expands or shrink the spinner will change with it.</div>
  </div>
  <div class="loading-container">
   <div class="loading-raygun"></div>
  </div>
 </div>
</div>
