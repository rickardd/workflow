2.1 #permalink Element structure

  • parent
    parent element
  • parent__child
    child element
  • parent__child__child
    grand child element

2.2 #permalink Properties and stats

  • parent--property
    parent property
  • parent__child--property
    child property
  • parent__child__child--property
    grand child property
