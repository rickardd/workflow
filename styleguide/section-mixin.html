13 #mixin Mixins

Mixins can be used anywhere with @include

13.1 #mixin.1 transition($property)

  • $property
    Any rule to prefix

Adds browser prefixes with default values "all 200ms linear"

Compatible in ???

13.2 #mixin.2 translate($translate)

  • $translate
    The translate property that will be added to the transform rule.

Adds browser prefixes. E.g -webkit-transform: scale(2);

Compatible in ??

13.3 #mixin.3 transform()

Adds browser prefixes to following properties

transform:translateZ(0); transform-style: preserve-3d; transform: translate3d(0, 0, 0); backface-visibility: hidden;

Compatible in ??

13.4 #mixin.4 clearfix()

Stops the container to collapse caused of floating children.

Compatible in ??

13.5 #mixin.5 touch-scroll()

Gives touch screens a momentum basted scroll. When scrolling with finger the page will continue to scroll for a while. The body has momentum scroll by default but this should be added to each element that has a overflow-y: scroll.

Compatible in only webkit browsers and is not a standardized property.

13.6 #mixin.6 verticalGradient($start,$end)

Makes a linear gradient.

$start - start color + background colors for browsers that not supporting gradient. $end - end color

Compatible in ??

13.7 #mixin.7 scale($size:0.3)

browser prefixes the scale property

$size - default 0.3

Compatible in ??

13.8 #mixin.8 noselect()

prevent the dragging feature when dragging text.

Compatible in ??

13.9 #mixin.9 font-size($sizeValue: 1.6)

Gives the font size px ard rem values

Compatible in ??

13.10 #mixin.10 placeholder-color($color)

Changing the color of the placeholder text for text-input fields. Browser prefixed.

$color - The color of the text

Compatible in ??

13.11 #mixin.11 noWrap()

Adds 3 dots after text when in collapses. Wont brake to more than one line.

Compatible in ??

13.12 #mixin.12 @extend wordbreak

Breaks a line to multiple lines if it doesn't fit in it's parent.

Deprecated: make an @mixin for the same purpose.

Compatible in ??

13.13 #mixin.breakpoints Responsive breakpoints

  • $screen-max
    Smaller than 1400px
  • $screen-min
    Smaller than 840px
  • $mw-screen-min
    Smaller than 840px
  • $screen-med
    Smaller than 960px
  • $mw-screen-med
    Bigger than 960px
  • $screen-mob
    Smaller than 640px
  • $mw-screen-mob
    Bigger than 640px
