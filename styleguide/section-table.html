17 #table Tables

The generic table is created with the base class .table-standard

17.1 #table.1 NOTE!

Many tables have a width class e.g table__cell-120 defined on the td. That is NOT necessary if it is defined on the th element.

17.7 #table.7 Cell size and spacing

  • .table__cell--standard
    Normal spacing
  • .table__cell--tight
    Tight spacing
  • .table__cell--tightest
    Even tighter spacing
  • .table__cell--standard .apikey.hide
    ???
  • .table__cell--checkbox
    width and padding for stand alone checkboxes.
  • .table__cell--button
    width and padding for buttons.
  • .table__cell--message
    Adds the padding to the child link instead of the cell itself.

17.10 #table.10 Cell positioning

  • .table__cell--left
    aligns it's content to the left
  • .table__cell--right
    aligns it's content to the right
  • .table__cell--center
    center aligns it's content
