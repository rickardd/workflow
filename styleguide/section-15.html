15 #permalink Mixins

Mixins can be used anywhere with @include

15.1 #permalink transition($property)

  • $property
    Any rule to prefix

Adds browser prefixes

Compatible in IE6+, Firefox 2+, Safari 4+.

15.2 #permalink transform-transition($speed, $type)

  • $speed
    Default 200ms
  • $type
    Default ease-in-out

Adds browser prefixes to transform rules. e.g -webkit-transition: -webkit-transform $speed $type;

Compatible in IE6+, Firefox 2+, Safari 4+.
